The Brain

The Brain is a portable cloud environment designed to help with the data and analytics functions required to purchasing media. Included in your environment is a Jupyter notebook server per user along with a Postgres database with MADLIB.net extention.

Included in this deployment are the following tools:

  • A User Management system capable of monitoring user activity so as to insure our data is both protected and used correctly
  • Jupyter to write notebooks for: R, Python & Julia
  • Code Templates for:
    • Channel Optimizer
    • Reach Optimizer
    • Topic Model
    • Customer Life Time Value Model
    • Customer Recency Frequency & Monetary Segmentation
    • Audience Building
    • Customer Journeys
    • Recommendation Engine
  • Database with support for in database analytics and machine learning using MADLIB.net
